🎬 JVS.DNG — Filmmaker & Editor Portfolio

Um portfólio moderno e imersivo para filmmaker/editor, com foco em experiência visual premium, animações suaves e design minimalista.

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

🚀 Tecnologias Utilizadas
	•	HTML5
	•	CSS3 (Responsivo + Glass UI)
	•	JavaScript (Vanilla)
	•	GSAP (GreenSock Animation Platform)
	•	ScrollTrigger
	•	Lenis (Smooth Scroll)

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻


✨ Funcionalidades
	•	 Hero com vídeo em background (cinematográfico)
	•	 Smooth Scroll com Lenis
	•	 Animações avançadas com GSAP
	•	 ScrollTrigger (animações ao rolar a página)
	•	 Cursor customizado interativo
	•	 Preloader animado com contador
	•	 Destaque dinâmico de palavras (Expertise)
	•	 Dropdown de projetos (clicável + animado)
	•	 Efeito magnético em CTA
	•	 Layout totalmente responsivo (mobile + desktop)

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻


📱 Responsividade

O projeto foi otimizado para:
	•	 Dispositivos móveis
	•	 Desktop
	•	 Diferentes tamanhos de tela (Ainda em desenvolvimento)

  ⚙️ Como Rodar o Projeto

🔹 Opção 1 — Live Server (Recomendado)
	1.	Instale a extensão Live Server no VS Code
	2.	Clique com botão direito no index.html
	3.	Clique em Open with Live Server
